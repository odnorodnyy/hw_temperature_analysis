# Анализ температурных данных 

Домашняя работа по реализации интерактивного веб приложение для анализа данных о температуре и мониторинга текущей погоды через `OpenWeatherMap API`.

## Функции
- Загрузка данных о температуре из `CSV` файла
- Вычисление скользящего среднего и стандартного отклонения
- Выявление аномалий по правилу $mean \pm 2\sigma$ 
- Сезонные профили и месячная статистика
- Получение текущей температуры
- Сравнение текущей температуры с историческими нормами

## Стек
```
- Python
- Streamlit
- Pandas
- Plotly
- OpenWeatherMap API
```

## P.S.
- Для получения текущей погоды необходим `API` ключ `OpenWeatherMap`
- Анализ данных реализован в двух вариантах: последовательный и параллельный
- Запросы к `API` реализованы синхронно (`requests`) и асинхронно (`aiohttp`)
